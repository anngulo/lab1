Question 3. RAW: Find out what each displayed number means, explain in the report.

MCC - Mobile Country Code to identify the country which a mobile phone belongs to

MNC - Mobile Network Code to identify a mobile phone's network

TAC - Tracking Area Code to identify a tracking area in a particular network

ECI - E-UTRAN Cell Identifier to identify cells globally

PCI - Physical Cell ID to indicate the physical layer identity of the cell

LCID - Local Cell Identifier

NID - Network ID

ASU - Arbitrary Strength Unit is an integer value that is proportional to the received signal strength

RSRQ - Reference Signal Received Quality to indicate the quality of the received reference signal

RSSNR - Reference Signal Signal to Noise Ratio is the ratio of signal power to the noise power

CQI - Channel Quality Indicator is an indicator which shows how good or bad the communication channel quality is

TA - Timing Advance allows to control the signal delays between the Mobile Station (MS) and Base Transceiver Station (BTS)

RSRP - Reference Signal Received Power shows the average power of the received pilot signals or the level of the received signal from the Base Station.

Power shows signal power in watts

Band shows frequency spectrum
